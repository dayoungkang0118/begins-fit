# 비긴핏 MVP

비긴에스가 지식산업센터 부동산을 통해 유입되는 오피스 입주 검토 고객에게 무료 테스트핏을 제공하고, 이후 인테리어 견적으로 전환하기 위한 1차 웹앱입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

## 포함된 흐름

- 고객 직접 테스트핏 신청
- 소개 부동산 자동 기록
- 소개 부동산 URL 자동 기록
- 비긴에스 회사 및 전문 분야 소개
- 도면 확보 방법 선택
- PDF, CAD, 이미지 도면 첨부
- 업무 좌석 수와 책상 크기 입력
- 회의실 인원 및 개수 입력
- 회사의 공간 사용 방식 체크
- 비긴에스 관리자용 요청 목록
- 진행 상태 관리
- 브라우저 로컬 저장소 기반 데이터 저장

## 운영 원칙

부동산이 비긴에스에 무제한으로 테스트핏을 요청하는 방식이 아니라, 부동산이 고객에게 링크를 전달하고 실제 입주 검토 고객이 직접 신청하는 방식입니다.

기본 무료 테스트핏은 1회 제공을 전제로 하고, 여러 호실 반복 검토나 대량 비교는 별도 협의 대상으로 분리하는 것이 좋습니다.
